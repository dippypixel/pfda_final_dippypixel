## Description
Breakout:Force is a breakout clone that differentiates itself by combining aspects of air hockey and physics. This game challenges your precision, reaction time, and motor skills.
### Gameplay:
    The Striker is controlled by the mouse, and you can hit the ball with it. The ball reacts accordingly to how much force you hit the ball with and the direction that you hit it from. The ball can also bounce off of the walls, and from the blocks themselves.
    If you hit the ball fast enough, the ball will be "explosion-ready", which is indicated by the rendered after-images and the texture change. Once the ball comes in contact with the blocks, A big explosion happens which eliminates more blocks than the ball would on its own, adding more depth to the game play.
    You start off with six lives, and the more lives you lose, the more red the background becomes as a visual indicator about how close you are to losing.
### Other Functionality:
    The game has a functional, though minimalistic title screen, win screen, and lose screen, and a functional restart mechanic whenever the game is over to play again when the player wants to.
### File Description:
    This game contains the source files for the sprites and sounds used in the game, along with the python file itself.
### Future areas of improvement:
    The visuals of the game, besides the handdrawn sprites, are pretty barren and I feel like I could've expanded upon the background or the title/win/lose screens. 
    Game feel is not entirely up to my original vision, I intended to add pause screenshake whenever the ball is hit very hard, but that felt out of my scope. 
### Considerations:
    I did consider adding barriers to the corners on the bottom like real air hockey, but that was turned down due to decreasing the difficulty and lessening the impact of when you win.
### Challenges:
    The challenges I faced when creating this project are:
    -learning how to code basic physics
    -how to code collision
    -how to render images as sprites
    -how to play sound 
    -how to create various screens
